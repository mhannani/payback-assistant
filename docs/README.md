# Documentation

Engineering notes for the assistant, kept alongside the code.

- **[decisions/](decisions/)** — Architecture Decision Records (ADRs): the non-obvious
  design choices, why they were made, the alternatives considered, and the references
  that informed them. Each record is dated and self-contained.

The goal is that a reader can understand not just *what* the code does, but *why* it is
built this way — especially where a naive approach would have been subtly wrong.
